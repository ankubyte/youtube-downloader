# 📥 YouTube Downloader

A simple and fast YouTube video downloader built with Python using the powerful `yt-dlp` library.

---

## ✨ Features

- Download YouTube videos in various qualities
- Simple and lightweight — no bloat
- Built on `yt-dlp`, one of the most reliable YouTube download libraries

---

## 🛠️ Requirements

- Python 3.7+
- `yt-dlp` module

---

## 📦 Installation

**1. Clone the repository**

```bash
git clone https://github.com/ankubyte/youtube-downloader.git
cd youtube-downloader
```

**2. Install the required dependency**

```bash
pip install yt-dlp
```

---

## 🚀 Usage

```bash
python yt-download.py
```

Enter the YouTube URL when prompted and the video will be downloaded to your current directory.

---

## 📚 Dependencies

| Package | Purpose |
|--------|---------|
| `yt-dlp` | Core library for downloading YouTube videos |

Install all dependencies with:

```bash
pip install -r requirements.txt
```

> `requirements.txt`
> ```
> yt-dlp
> ```
